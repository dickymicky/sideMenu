### Options

- `selector` 사이드 메뉴의 선택자
- `showBtnSelector` 사이드 메뉴 활성화 버튼 선택자
- `shadowSelector` 사이드 메뉴 활성화 배경

```js
var sideMenu = new SideMenu({
    selector: '.side_menu',
    showBtnSelector: '.side_menu_btn',
    shadowSelector: '.side_shadow'
});
```
